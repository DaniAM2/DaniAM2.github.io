HTML5

### ¿Para qué sirve HTML5?
* HTML5 es un lenguaje de marcado que nos permite crear y estructurar el contenido de nuestra web; * Nos ayuda a establecer una jerarquía semántica que facilita el trabajo con categorías y secciones distintas, por ejemplo para crear nuestro propio blog;
### ¿En qué consiste HTML5?
1. Para crear un archivo HTML necesitaremos únicamente nuestros conocimientos y un editor de textos cualquiera (preferiblemente uno como Sublime); 2. HTML se basa en un sistema de etiquetas con cierre. Para cerrar la etiqueta inluiremos un / dentro de los símbolos de etiquetado; 3. Cada una de las etiquetas deﬁne un elemento concreto y **estandarizado**, como pueden ser imágenes o encabezados de párrafo

Atributos

¿Qué son y para qué sirven?

Los atributos en HTML deﬁnen distintos valores para las etiquetas, desde identiﬁcadores hasta tamaños de imagen. Aquí se representan algunos de ellos:

Etiqueta Atributo Función
p class = " " aplica un estilo definido
img src = " " define ruta de imagen
img alt = " " descripción (accesibilidad)

La imagen superior representa cómo se vería en un navegador nuestro HTML sin estilos, y la inferior tiene unos estilos muy simples. ¿Notas la diferencia?

  1. Crear un archivo con extensión .css en nuestro editor de textos;
  2. Guardarlo en la misma carpeta que nuestro HTML;
  3. Referenciarlo en nuestra cabecera del HTML;
